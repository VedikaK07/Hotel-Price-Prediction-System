# 🤖 ML Prediction Project

A Machine Learning web application built using Flask and Python for predictive analysis and smart recommendations.

## 🚀 Features
- Machine Learning predictions
- Interactive web interface
- Flask backend
- Multiple ML models
- User-friendly UI

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy

## ▶️ Run Project
pip install -r requirements.txt

python app.py
